ikastetxearen datuekin eta eskuragarri izan beti — inprimatutako kopia bat
eta bertsio digitala leku seguru batean.

## 3. Ekintzak

### 3.1 Edukiera (lehen orduan)

- [ ] Kaltetutako gailua saretik isolatu
- [ ] Konprometitutako kontuak blokeatu
- [ ] Kaltetutako zerbitzuak gelditu (beharrezkoa bada)
- [ ] Jakinarazi koordinatzaileari eta IT arduradunari
- [ ] Jakinarazi komunikazio-arduradunari

### 3.2 Ikerketa (24–72 ordu)

- [ ] Zehaztu zer gertatu den (eraso-bektorea)
- [ ] Identifikatu kaltetutako aktiboak eta datuak
- [ ] Babeskopien egoera egiaztatu
- [ ] Frogak gorde (log-ak, pantaila-argazkiak…)

### 3.3 Berreskurapena

- [ ] Kaltetutako sistemak babeskopietatik berreskuratu
- [ ] Pasahitzak aldatu sistema guztietan
- [ ] Egiaztatu sistemak garbi daudela berriro konektatu aurretik
- [ ] Zerbitzuak berrezarri lehentasun-ordenaren arabera

### 3.4 Jakinarazpen ofizialak

- [ ] AEPD (datu-urraketa bada — 72 orduko epean)
- [ ] Cyberzaintza (intzidentzia teknikoa bada)
- [ ] Beste agintaritzak (ENS, NIS2…)
