# Arriskuen Analisiaren Txantiloia

Arriskuen analisiaren txantiloia, Margerit metodologian oinarritua. Bete atal bakoitza
zure ikastetxearen testuinguruarekin.

---

## 2. Mehatxuen identifikazioa

| Mehatxua | Deskribapena | Eragindako aktiboa(k) | Probabilitatea (Altua/Ertaina/Baxua) |
|----------|-------------|----------------------|--------------------------------------|
| | | | |

Mehatxu arruntak:

- **Ransomware** — Fitxategiak zifratu eta erreskatea eskatu
- **Phishing** — Posta bidez pasahitzak edo datuak lapurtu
- **Datu-filtrazioa** — Informazio sentikorra nahi gabe edo nahita partekatu
- **Ekipamendu lapurreta** — Gailuak lapurtu, datuak arriskuan
- **Sarearen erorketa** — Internet / zerbitzuak etetea
- **Barne-akatsa** — Langile batek nahigabe datuak ezabatu edo konfigurazioa aldatu
- **Sarbide ez-baimendua** — Baimenik gabeko norbait sistemara sartu
- **Software ahultasuna** — Eguneratu gabeko softwarearen bidez erasoa
- **Hornikuntza-katearen erasoa** — Hornitzaile baten bidez sartzea

## 3. Arriskuen ebaluazioa

| Arriskua | Probabilitatea | Inpaktua | Arrisku-maila (P x I) | Lehentasuna |
|----------|---------------|----------|----------------------|-------------|
| | | | | |

### Balorazio-eskala

- **Probabilitatea:** 1 (oso baxua) — 5 (oso altua)
- **Inpaktua:** 1 (txikia) — 5 (kritikoa)
- **Arrisku-maila** = Probabilitatea × Inpaktua
- **Lehentasuna:**
  - 15–25: **Kritikoa** — berehalako neurria
  - 8–14: **Altua** — epe laburrean
  - 4–7: **Ertaina** — programatu
  - 1–3: **Baxua** — onartu edo monitorizatu

## 4. Neurrien plana

| Arriskua | Neurria | Mota | Ard. | Epea | Egoera |
|----------|---------|------|------|------|--------|
| | | [Saihestu / Murriztu / Transferitu / Onartu] | | | [Egiteke / Martxan / Eginda] |

---

Txantiloi hau moldatu zure ikastetxearen errealitatera. Arrisku bakoitzarentzat
aztertu ea neurria **saihestu** (ez egin arriskua sortzen duen jarduera), **murriztu**
(segurtasun-neurriak ezarri), **transferitu** (asegurua kontratatu, zerbitzua
azpikontratatu) edo **onartu** (arriskua onargarria bada eta neurriak baino
garestiagoa ez bada) nahi duzun.

> Iturri ofizialak: CCN-CERT Margerit metodologia, Cyberzaintza Arriskuen analisi gida.
