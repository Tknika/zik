# RACI Matrizea — Zibersegurtasuneko rolak

Zibersegurtasuneko gobernantzan parte hartzen duten rol eta erantzukizunak definitzeko RACI matrizea.

**R = Responsible** (arduraduna) · **A = Accountable** (erantzulea) · **C = Consulted** (kontsultatua) · **I = Informed** (informatua)

---

| Zibersegurtasun politika onartzea | A | R | C | C | I |
| Urteko segurtasun plana egitea | A | R | C | C | I |

| Sentsibilizazio kanpainak antolatzea | I | R | C | R | I |
