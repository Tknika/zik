---
title: Glosarioa
description: Zibersegurtasuneko eta lege-testuetako terminoak euskaraz
---

Zibersegurtasunarekin eta lege-araudiarekin lotutako termino garrantzitsuenak,
euskaraz azalduta eta gaztelaniazko baliokidearekin.

---

| Euskara | Gaztelania | Azalpena |
|---------|-----------|----------|
| **Ahultasuna** | Vulnerabilidad | Sistema edo aplikazio batean erasotzaile batek ustia dezakeen akatsa edo ahulezia. |
| **Aktiboa** | Activo | Ikastetxearentzat balioa duen edozein elementu fisiko edo digital: zerbitzariak, datuak, softwarea, pertsonak… |
| **Arriskua** | Riesgo | Mehatxu batek ahultasun bat ustiatzean kaltea eragiteko probabilitatea eta haren inpaktua. |
| **Autentikazio-faktore anitz (MFA)** | Autenticación multifactor | Erabiltzaile baten identitatea egiaztatzeko bi faktore edo gehiago eskatzen dituen segurtasun-mekanismoa. |
| **Babeskopia** | Copia de seguridad | Datuen kopia bat, jatorrizkoak galtzen badira berreskuratzeko. |
| **Ciberseguridad** | Zibersegurtasuna | Zibersegurtasuna (gaztelaniazko terminoa erreferentzia gisa). |
| **DPD (Datuen Babeserako Ordezkaria)** | DPD (Delegado de Protección de Datos) | RGPDk eskatzen duen figura: erakunde batean datuen babesa gainbegiratzen duen pertsona. |
| **EIPD (Datuen Babesari buruzko Eraginaren Ebaluazioa)** | EIPD / DPIA | Datuen tratamendu batek pertsonen eskubide eta askatasunetan izan ditzakeen arriskuak ebaluatzeko prozesua (RGPD). |
| **ENS (Esquema Nacional de Seguridad)** | ENS | Espainiako sektore publikoko informazio-sistemen segurtasuna arautzen duen araudia (311/2022 ED). |
| **Eraldi taldea** | Equipo Eraldi | Ikastetxeetako eraldaketa digitalean diharduen taldea, Tknikak bultzatua. |
| **Erresilientzia** | Resiliencia | Sistema batek intzidentzia baten ondoren normaltasunera itzultzeko eta funtzionatzen jarraitzeko gaitasuna. |
| **Firewall (suebakia)** | Cortafuegos | Sareko trafikoa iragazten duen hardware edo softwarea, baimendutako konexioak soilik onartuz. |
| **Gobernantza** | Gobernanza | Zibersegurtasunaren kudeaketarako erabakiak, rolak eta politikak definitzen dituen prozesu eta egituren multzoa. |
| **Intzidentzia** | Incidente | Segurtasuna arriskuan jartzen duen gertaera: datu-filtrazioa, ransomware erasoa, sarbide ez-baimendua… |
| **ISO/IEC 27001** | ISO/IEC 27001 | Informazioaren segurtasuna kudeatzeko nazioarteko estandarra (ISMS). |
| **ISMS (Informazioaren Segurtasuna Kudeatzeko Sistema)** | SGSI (Sistema de Gestión de Seguridad de la Información) | ISO/IEC 27001 estandarraren muina: erakunde batek bere informazioaren segurtasuna kudeatzeko ezartzen duen politika, prozedura, kontrol eta arriskuen kudeaketaren multzo sistematikoa. |
| **LOPDGDD** | LOPDGDD | Espainiako Datu Pertsonalen Babeserako eta Eskubide Digitalen Bermerako Lege Organikoa (3/2018 LO). |
| **Magerit** | Magerit | CCN-CERTek garatutako arriskuen analisirako metodologia, ENS betetzeko erabilia. |
| **Mehatxua** | Amenaza | Sistema edo datuentzako kalte potentziala eragin dezakeen gertaera, nahita edo nahigabe. |
| **NIS2** | NIS2 | Europar Batasuneko Zibersegurtasun Zuzentaraua (2022/2555), sektore kritikoen segurtasun-neurriak arautzen dituena. |
| **NIST CSF 2.0** | NIST CSF 2.0 | AEBko NISTek argitaratutako Zibersegurtasun Markoa, sektore guztietako erakundeentzat baliagarria. |
| **Pasahitzen kudeatzailea** | Gestor de contraseñas | Pasahitzak modu seguruan sortu, gorde eta betetzen dituen aplikazioa (Bitwarden, KeePass…). |
| **Phishing** | Phishing | Posta elektroniko, SMS edo webgune faltsuen bidez datu pertsonalak (pasahitzak, banku-datuak) lapurtzeko teknika. |
| **RACI matrizea** | Matriz RACI | Rolak eta erantzukizunak definitzeko matrizea: Responsible, Accountable, Consulted, Informed. |
| **Ransomware** | Ransomware | Fitxategiak zifratu eta berreskuratzearen truke erreskatea eskatzen duen software maltzurra. |
| **RGPD (DBEO)** | RGPD | Europar Batasuneko Datu Pertsonalen Babeserako Erregelamendu Orokorra (2016/679 EB). |
| **RTO** | RTO (Recovery Time Objective) | Zerbitzu bat berreskuratzeko onargarria den gehienezko denbora, intzidentzia baten ondoren. |
| **Segurtasun-politika** | Política de seguridad | Erakunde batek bere segurtasun-ikuspegia, arauak eta ardurak definitzen dituen dokumentua (ENS Art. 12). |
| **SIEM** | SIEM | Segurtasun-gertaeren kudeaketa zentralizaturako sistema (erregistroak bildu, aztertu eta alertak sortzen ditu). |
| **VLAN** | VLAN | Sare lokala segmentatzeko teknologia, trafikoa eta sarbideak gune logiko desberdinetan banatuz. |
| **WiFi** | WiFi | Haririk gabeko sare lokala (WLAN). |
| **Zifraketa** | Cifrado | Datuak irakurtezin bihurtzeko teknika; gakoa duenak soilik deszifra ditzake. |
| **3-2-1 araua** | Regla 3-2-1 | Babeskopien oinarrizko printzipioa: 3 kopia, 2 euskarri desberdin, 1 kokapen fisiko desberdinean. |

::tip[Terminoen bila?]
Erabili bilatzailea (sakatu `/` zure teklatuan) edozein termino azkar aurkitzeko.
::
